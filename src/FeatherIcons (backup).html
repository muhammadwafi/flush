How to use
To use this icon see some description below.

To use an icon on your page, add a data-feather attribute with the icon name to an element. Example: 

<i data-feather="activity"></i>

You can also add a class to change the icon size or to customize the icon. The class list available are listed below.

SVG icon sizing

To change icon size use .icon-* class. Available icon sizing class for svg icon are:

.icon-xs
.icon-sm
.icon-nm
.icon-nm
.icon-md
.icon-lg
.icon-xl
SVG icon theming

To customize icon colors use .fill-* class. Available icon theming class for svg icon are:

default color
.fill-primary
.fill-secondary
.fill-light
.fill-dark
.fill-success
.fill-danger
.fill-warning
.fill-info
.fill-none
